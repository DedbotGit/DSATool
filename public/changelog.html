Changelog - DSA Tool



v0.3.3 Beta (April 24 2024)


  • -Added the Blueprint Cost page which shows the cost of any blueprint in raw materials and specific items, it also shows the size of the blueprint
  • -Hopefully fixed a bug with the mobile menu on the changelog page



v0.3.2 Beta (Febuary 5 2024)


  • -Added the resources page which shows various resources for the game
  • -Added the accent color to more places
  • -Added a flux to item conversion in the exchange rates
  • -Refined visual effects
  • -Added partnerships to the resources page
  • -Optimized exchange rates to load much faster
  • -Made the economy viewer mobile-friendly



v0.3.1 Beta (Febuary 2 2024)


  • -Added a mobile-friendly navigation bar
  • -Added custom link highlights and other visual improvements
  • -Adjusted item prices to be more accurate



v0.3.0 Beta (Febuary 1 2024)


  • -Added a new tool: Exchange Rates
  • -Exchange rates shows the current price for items by calculating it's usefulness and then multiplying that by the ratio of new flux to new items to calculate it's rarity.
  • -The prices are still a work in progress, if you think that they should be adjusted then go to the DSATool Discussions on Github
  • -Removed light mode for the time being, it may be re-added later
  • -Added a new navigation menu



v0.2.0 Beta (January 23 2024)


  • -Implemented a graph for the economy tool
  • -Removed the old text-based list for the economy tool
  • -Added a calculation for loaders needed per turret
  • -Added a description of DSATool to the homepage
  • -Added DSA-IP as an offical partener



v0.1.4 Beta (January 19 2024)


  • -Added a core consumption calculation
  • -Added a slider to determine what percent of maximum core consumption the core consumption calculation uses (go to zero for idle consumption calculation)
  • -Added a conversion between ship blocks, rcs, and world blocks
  • -Added a total damage per second calculation



v0.1.3 Beta (January 17 2024)


  • -Fixed the positioning of the dark mode button (the dark mode header is a work in progress)
  • -Improved the calcuation for shield regen to take into account the shield leak (counts all shields as active)



v0.1.2 Beta (January 16 2024)


  • -Added a dark mode (still a work in progress)
  • -Improved formatting for Item Drops
  • -Other minor fixes



v0.1.1 Beta (January 15 2024)


  • -Added a new webpage: Item Drops (not a finalized name)
  • -Bot drops shows only the bot drops txt file but this will likely be expanded upon with adding images of the bots and their stats, in addition to other sources of loot like pits and canary



v0.1.0 Beta (January 11 2024)


  • -Moved to a new website: dsatool.com
  • -Separated the website into different pages including: the home page, the calculator tool, the economy data dumps viewer, and the changelog.
  • -Fixed a bug that prevented inputting single numbers for ship size in the calculator.
  • -Made non-input areas more clear in the calculator.
  • -Added navigation between pages in the website.
